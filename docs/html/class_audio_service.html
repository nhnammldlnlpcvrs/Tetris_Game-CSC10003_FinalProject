void AudioService::Init ( )

Khởi tạo hệ thống âm thanh và load các file âm thanh/nhạc nền.

Khởi tạo hệ thống âm thanh và load các hiệu ứng + nhạc nền.

  • Gọi InitAudioDevice().
  • Load hiệu ứng xoay khối, xóa dòng.
  • Load và phát nhạc nền.
  • Gọi InitAudioDevice() để khởi tạo thiết bị âm thanh.
  • Load âm thanh xoay khối, âm thanh xóa dòng, và nhạc nền.
  • Tự động phát nhạc nền ngay khi khởi tạo.
