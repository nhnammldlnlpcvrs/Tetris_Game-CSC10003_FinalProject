Detailed Description

Quản lý và điều khiển hành vi của một Block trong trò chơi.

BlockController chịu trách nhiệm di chuyển, xoay khối và quản lý undo/redo thông qua việc lưu lại lịch sử trạng thái (Memento Pattern). Ngoài ra, nó áp dụng Command Pattern để thực hiện và đảo ngược các hành động.
