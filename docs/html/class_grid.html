Note
Dùng Color tự định nghĩa, không phải raylib::Color
Dùng Position tự định nghĩa, không phải raylib::Vector2

Đại diện cho lưới (bảng trò chơi) chứa các cell màu sắc.

  • Mỗi cell có thể rỗng hoặc chứa màu (tương ứng với khối).
  • Quản lý việc kiểm tra hợp lệ, xóa dòng đầy, và trạng thái lưới.
