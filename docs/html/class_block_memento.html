Detailed Description

Lưu trạng thái xoay và vị trí của khối Block để hỗ trợ undo/redo.

Áp dụng trong Memento Pattern, lớp này đóng vai trò "Memento", lưu giữ trạng thái (rotation, row, column) của một Block.
