Detailed Description

Lớp chịu trách nhiệm hiển thị lưới (grid) và các ô chứa block.

Lớp này đảm nhiệm vai trò "View" trong mô hình MVC, chỉ phụ trách phần hiển thị, không xử lý logic của game. Nó sẽ vẽ:

  • Đường viền lưới với màu xám nhạt.
  • Các ô block đã được tô màu trong grid.
