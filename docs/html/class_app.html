void App::Run ( )

Hàm chạy ứng dụng chính.

Main application loop for the Tetris game.

Khởi tạo cửa sổ, tải tài nguyên, chạy vòng lặp game cho đến khi thoát.

Initializes the window, sets the target FPS, loads resources, and runs the main game loop until the window is closed. Inside the loop, it handles:

  • Updating the game state
  • Drawing game board, next block preview, score, and guide
  • Handling game over state and restart

Load game font from assets
