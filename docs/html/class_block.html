Detailed Description

Lớp đại diện cho một khối trong trò chơi (Tetris Block).

Lớp này quản lý vị trí, trạng thái xoay, màu sắc và khả năng vẽ khối. Đồng thời hỗ trợ lưu/khôi phục trạng thái theo Memento Pattern.
