Detailed Description

Đại diện cho khối T trong trò chơi Tetris.

Khối T có bốn trạng thái xoay khác nhau, với tâm xoay ở vị trí trung tâm. Mỗi khối được xác định bởi màu sắc và tọa độ các ô trong ma trận.
