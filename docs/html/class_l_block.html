Detailed Description

Lớp đại diện cho khối L trong trò chơi Tetris.

Kế thừa từ lớp Block và cài đặt các phương thức đặc thù của khối L:

  • Xác định hình dạng (các trạng thái xoay).
  • Xác định màu sắc cho các ô (cells).
  • Cho phép nhân bản đối tượng (Clone).

Khởi tạo khối L với:

  • ID = 2
  • Trạng thái xoay ban đầu = 0
  • Vị trí offset hàng/cột = 0
  • Màu cam (orange)
  • Các cấu hình tọa độ của 4 trạng thái xoay.
