Detailed Description

Quản lý toàn bộ vòng đời và logic chính của trò chơi Tetris.

Lớp này xử lý việc tạo khối, di chuyển, xoay, tính điểm, khóa khối và kiểm tra trạng thái trò chơi. Đồng thời chịu trách nhiệm giao tiếp với dịch vụ âm thanh và vẽ màn hình.
